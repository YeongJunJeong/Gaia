# Gaia 풍부도 복원 벤치마크 — 2026년 4월 2일

---

## 1. 과제 설명

### 로드맵 원본 (과제 5)
```
입력: 20%/30%/50% 마스킹된 프로필
출력: 원래 프로필 복원
지표: 코사인 유사도, MAE
```

### 실제 구현: 과제를 모델 구조에 맞게 수정

MGM/Gaia는 GPT-2 기반 Causal LM으로, **앞→뒤 한 방향만 예측**하는 모델이다.
로드맵의 "마스킹 복원"은 양방향 모델(BERT)에 적합한 과제이므로,
GPT 스타일에 맞는 **"다음 토큰 예측"** 방식으로 변경했다.

```
마스킹 복원 (BERT용): [A] [?] [C] [?] [E] → B, D를 맞춰라
다음 토큰 예측 (GPT용): [A] [B] [C] → D, E를 맞춰라
```

실제로 마스킹 복원을 먼저 시도했으나, MGM 원본도 Top-1 1.3%로 
모델 구조의 한계가 확인되어 과제를 수정함.

---

## 2. 실험 설계

- 토양 미생물 목록을 풍부도 순으로 정렬
- 앞부분 (80%, 70%, 50%)을 모델에 입력
- 나머지 (20%, 30%, 50%)를 모델이 순서대로 예측
- 예측한 미생물이 실제 미생물과 일치하는지 측정

### 비교 대상
- **MGM 원본**: 26만 샘플(모든 바이옴)로 학습된 모델
- **Gaia**: MGM + 토양 2,887개로 추가 학습한 모델

### 테스트 데이터
- MGnify 토양 데이터 중 학습에 사용하지 않은 **200개 샘플**

## 4. 의미

### Gaia > MGM 원본
- MGM은 26만 샘플(모든 바이옴)로 학습했지만, 토양 예측에서는 Gaia가 우수
- 토양 특화 추가 학습(2,887개)이 효과적

### Random Forest가 할 수 없는 과제
- RF는 "빠진 미생물 예측" 자체가 불가능
- 이 과제는 파운데이션 모델만의 고유 능력

### Top-10 기준 31.3%의 의미
- 9,669종 미생물 중 상위 10개 안에 정답이 31.3% 확률로 들어있음
- 랜덤 기대값은 0.1% (10/9669)
- **랜덤 대비 313배 정확**
